# GitHub Explorer

A GitHub profile search app built with React, TypeScript, and Tailwind CSS.

## Features

- Search any GitHub username
- View profile info: followers, following, repos, bio, location
- Browse top repositories with language, stars, and forks
- Loading spinner and error handling
- Fully responsive dark theme

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router
- GitHub REST API
- Vite
